YAPILICAKLAR LİSTESİ

Bu proje, günlük yapılacak işleri kolayca takip edebilmek için hazırlanmıştır. Kullanıcı görev ekleyebilir, düzenleyebilir, silebilir ve tamamlanan görevleri işaretleyebilir. Tüm veriler Local Storage kullanılarak tarayıcıda saklanır, bu sayede sayfa yenilense bile görevler kaybolmaz.

ÖZELLİKLER

- Görev ekleme
- Görev düzenleme
- Görev silme
- Görevleri tamamlandı olarak işaretleme
- Local Storage ile verileri kaydetme
- Toplam görev sayısını gösterme
- Tamamlanan görev sayısını gösterme
- Tüm görevleri tek tıkla silme
- Enter tuşu ile hızlı görev ekleme

KULLANILAN TEKNOLOJİLER

- HTML
- CSS
- JavaScript
- Local Storage

PROJEYİ ÇALIŞTIRMA

1. Proje dosyalarını bilgisayarına indir.
2. `index.html` dosyasını tarayıcıda aç.
3. Görevlerini eklemeye başlayabilirsin.
